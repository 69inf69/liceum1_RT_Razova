# Random Password Generator

**Автор:** [Разова Дарья Сергеевна]

## Описание

Random Password Generator — это десктопное приложение для генерации надёжных случайных паролей с настраиваемыми параметрами. Программа позволяет:

* задавать длину пароля (от 4 до 128 символов);
* выбирать типы символов для генерации (буквы, цифры, специальные символы);
* копировать сгенерированный пароль в буфер обмена;
* просматривать историю генерации паролей;
* сохранять историю между запусками приложения.

Приложение разработано с использованием Python и библиотеки Tkinter для графического интерфейса. Данные сохраняются в формате JSON.

## Функциональность

**Основные возможности:**

1. **Настройка параметров пароля:**
   * ползунок для выбора длины пароля (4–128 символов);
   * чекбоксы для выбора типов символов: буквы (A‑Z, a‑z), цифры (0‑9), специальные символы (!@#$%&*);
   * динамическое отображение текущей длины пароля.

2. **Генерация пароля:**
   * кнопка «Сгенерировать» запускает процесс создания пароля;
   * сгенерированный пароль отображается в специальном поле;
   * возможность скопировать пароль в буфер обмена одним кликом.

3. **История генерации:**
   * таблица с историей всех сгенерированных паролей;
   * для каждого пароля сохраняются: сам пароль, длина, типы использованных символов, временная метка;
   * история сохраняется между сеансами работы в файле `history.json`.

4. **Валидация данных:**
   * проверка, что выбран хотя бы один тип символов;
   * ограничения на длину пароля (минимум 4, максимум 128 символов).

## Пример 1: Генерация простого пароля

Установите длину 8 символов.
Включите «Цифры» и «Буквы».
Нажмите «Сгенерировать».
Результат: K7mN9pQ2

## Пример 2: Генерация сложного пароля

Установите длину 16 символов.
Включите все типы символов.
Нажмите «Сгенерировать».
Результат: H8@kL2#mP9$qR5&

## Тестирование

**Тест 1: Корректная генерация**

Настройки: длина 12, включены все символы.
Ожидаемый результат: пароль из 12 символов с цифрами, буквами и спецсимволами.

**Тест 2: Минимальная длина**

Настройки: длина 4, включены цифры и буквы.
Ожидаемый результат: пароль из 4 символов.

**Тест 3: Некорректный выбор**

Настройки: все чекбоксы отключены.
Ожидаемый результат: сообщение об ошибке.
